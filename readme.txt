== Description ==
LeadXForms is a powerful WordPress plugin for generating and managing lead forms on your website. Easily create customizable forms to capture leads, manage submissions, and boost your lead generation efforts.

== Installation ==
1. Upload the 'leadxforms' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings under 'LeadXForms' in the WordPress admin dashboard.

== Frequently Asked Questions ==
= How do I create a new lead form? =
1. Go to the LeadXForms menu in your WordPress admin.
2. Click "Add New"
3. Configure your form fields and settings.
4. Save the form, and you can now embed it on your site.
